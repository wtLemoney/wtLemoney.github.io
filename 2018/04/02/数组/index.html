那些你常常会忘的处理数组的函数

发表于 2018-04-02 | 分类于 数组 ， 函数
字数统计: 348 | 阅读时长 ≈ 2

  哈哈，看似是些简单的处理数组的函数，但是由于平时码的少，实习生笔试的时候走了弯路吃了亏啊，写的很慢呀。。。有些是忘记了，有些是参数和返回值记忆模糊了，所以多码码，总结总结。持续更新中…

  • 1. splice()
  • 2. slice()
  • 3. filter()
  • 4. map()
  • 5. reduce()
  • 6. every()
  • 7. some()

1. splice()

返回值为被删除的元素数组，原数组会改变

2. slice()

可从已有的数组中返回选定的元素,并不会修改数组。

3. filter()

返回新的数组,原数组不改变

4. map()

生成新的数组

5. reduce()

原数组不改变

6. every()

判断arr是否都满足x>=18的条件

7. some()

判断是否有满足条件的
