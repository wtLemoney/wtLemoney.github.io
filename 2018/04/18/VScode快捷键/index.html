VScode常用快捷键

发表于 2018-04-18 | 分类于 VScode编辑器
字数统计: 542 | 阅读时长 ≈ 2

1.编辑器与窗口管理

  • 新建文件：Ctrl+N
  • 打开一个新窗口：Ctrl+Shift+N
  • 关闭本窗口所有文件：Ctrl+Shift+W
  • 文件之间切换：Ctrl+Tab
  • 左中右 3 个编辑器的快捷键：Ctrl+1 Ctrl+2 Ctrl+3
  • 编辑器换位置， Ctrl+k然后按 Left或 Right

2.代码编辑

格式调整

  • 代码行缩进：Ctrl+[，Ctrl+]
  • 上下移动一行：Alt+Up，Alt+Down
  • 向上向下复制一行： Shift+Alt+Up,Shift+Alt+Down
  • 在当前行下边插入一行 ：Ctrl+Enter
  • 在当前行上方插入一行 ：Ctrl+Shift+Enter

光标相关

  • 移动到行首：Home
  • 移动到行尾：End
  • 选中当前行：Ctrl+i
  • 删除光标所在行:Ctrl+Shift+K
  • 删除光标右侧的所有字：Ctrl+Delete
  • 移动到文件结尾：Ctrl+End
  • 移动到文件开头：Ctrl+Home
  • 移动到定义处：F12
  • 定义处缩略图：只看一眼而不跳转过去 Alt+F12
  • 跳到匹配的括号：Ctrl+Shift+\
  • 选择从光标到行尾： Shift+End
  • 选择从行首到光标处：Shift+Home
  • 扩展/缩小选取范围： Shift+Alt+Left 和 Shift+Alt+Right
  • 多行编辑(列编辑)：Alt+Shift+鼠标左键，Ctrl+Alt+Down/Up
  • 选择当前字的所有出现：Ctrl + F2
  • 同时选中所有匹配： Ctrl+Shift+L
  • 下一个匹配的也被选中：Ctrl+D
  • 回退上一个光标操作：Ctrl+U
  • 切出块注释/* */：Shift+Alt+A
  • 切出上/下一个操作的文件：Alt + ←/→

查找替换

  • 查找：Ctrl+F
  • 查找下一个/上一个：F3 / Shift + F3
  • 查找替换：Ctrl+H
  • 整个文件夹中查找：Ctrl+Shift+F

显示相关

  • 全屏：F11
  • 侧边栏显/隐：Ctrl+B
  • 显示 Debug：Ctrl+Shift+D
  • 显示 Output：Ctrl+Shift+U

重构代码

  • 找到所有的引用：Shift+F12
  • 同时修改本文件中所有匹配的：Ctrl+F12
  • 重命名：比如要修改一个方法名，可以选中后按 F2，输入新的名字，回车，会发现所有的文件都修改了
  • 跳转到下一个 Error 或 Warning：F8
  • 跳转到上一个 Error 或 Warning：Shift+F8
  • 查看 diff： 在 explorer 里选择文件右键 Set file to compare，然后需要对比的文件上右键选择 Compare with file_name_you_chose
